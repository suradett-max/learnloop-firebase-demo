# Classroom OS · WRK

ระบบบริหารชั้นเรียน ม.2 โรงเรียนวัดไร่ขิงวิทยา สำหรับเช็คชื่อ ติดตามงาน บันทึกคะแนน และดูภาพรวมชั้นเรียน โดยเก็บข้อมูลถาวรใน Cloud Firestore

## ความสามารถหลัก

- แดชบอร์ดภาพรวม พร้อมสถิติการเข้าเรียน งาน และข้อมูลรายห้อง
- รายชื่อนักเรียน เพิ่มรายคน นำเข้า CSV, Google Sheets และชุดข้อมูลเดิม
- เช็คชื่อแยกวันที่ วิชา และห้อง พร้อมบันทึกทุกคนในครั้งเดียว
- งานและคะแนน รองรับหลายวิชา หลายห้อง และหลายชิ้นงาน
- ศูนย์ดาวน์โหลด CSV/ข้อมูลสำรอง และเอกสารพิมพ์แบบเป็นทางการ
- โปรไฟล์ครู เวลาแบบสด การแจ้งเตือน และหน้าต่างยืนยันแบบทันสมัย
- แยกข้อมูลตามผู้ใช้งาน Firebase Anonymous Authentication

## รูปแบบ CSV นักเรียน

```csv
number,studentId,name,grade,room
1,12345,เด็กชายตัวอย่าง นักเรียน,ม.2,8
```
